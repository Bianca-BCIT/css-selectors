CSS Selectors

CSS Selectors are patterns used in CSS to select and style HTML elements on a page.

what is it and what does it do

Simple Selectors

Selects elements based on ID, name or class. Universal selector selects all elements. Element selector selects an element. Class selector selects all elements with specified class. ID selector selects all elements with specified ID.

Multiple Selectors

Selects all the element(s) that have the specified ID or class of two or more classes or IDs and chains them.

Attribute Selectors

A way to target HTML elements based on the presence or value of an attribute. They let you apply styles to elements with specific attributes, or even specific values within those attributes.

Combinators

Explains the connection between selector. There are 4 different types of combinators: Descendant combinator (space), Child combinator (>), Next sibling combinator (+), and Subsequent-sibling combinator (~).

Pseudo-Elements

A keyword added to a selector, allowing you to style a specific section of the chosen element(s).
