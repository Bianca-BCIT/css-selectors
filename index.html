CSS Selectors

CSS Selectors are patterns used in CSS to select and style HTML
elements on a page.

what is it and what does it do

CSS Simple Selectors

A CSS Simple Selector selects elements based on id, name or class. Universal selector - selects all elements. Element selector - selects element. Class selector - selects all elements with specified class. ID selector - selects all elements with specified ID.

CSS Multiple Selectors

CSS Multiple Selector selects all the element(s) that have the specified ID or class of two or more classes or IDs and chains them.

CSS Attribute Selectors

CSS Attribute Selectors are a way to target HTML elements based on the presence or value of an attribute. They let you apply styles to elements with specific attributes, or even specific values within those attributes. 

CSS Combinators

A combinator explains the connection between selectors. There are 4 different types of combinators:
Descendant combinator (space)
Child combinator (>)
Next sibling combinator (+)
Subsequent-sibling combinator (~) 

CSS Pseudo-elements

A CSS pseudo-element is a keyword added to a selector, allowing you to style a specific section of the chosen element(s).

CSS Pseudo-classes

Pseudo-classes selector is a selector that selects elements in a specific state.  For instance, when they are being hovered by the mouse pointer.
